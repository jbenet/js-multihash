Intro

Installable via npm install --save multihashes, it can also be used directly in the browser.

Quick Example

/* eslint-disable no-console */
'use strict'

const multihash = require('multihashes')
const buf = Buffer.from('0beec7b5ea3f0fdbc95d0dd47f3c5bc275da8a33', 'hex')

const encoded = multihash.encode(buf, 'sha1')
console.log(encoded)
// => <Buffer 11 14 0b ee c7 b5 ea 3f 0f db c9 5d 0d d4 7f 3c 5b c2 75 da 8a 33>

multihash.decode(encoded)
// => {
//      code: 17,
//      name: 'sha1',
//      length: 20,
//      digest: <Buffer 0b ee c7 b5 ea 3f 0f db c9 5d 0d d4 7f 3c 5b c2 75 da 8a 33>
//    }

There are more available, so take a look at the docs below for a full list. This documentation aims to be comprehensive, so if you feel anything is missing please create a GitHub issue for it.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save multihashes

You can then require() multihashes as normal:

const multihashes = require('multihashes')

In the Browser

multihashes should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of multihashes, including index.js and index.min.js, are included in the /dist folder. multihashes can also be found on unpkg.com under

  • https://unpkg.com/multihashes/dist/index.min.js
  • https://unpkg.com/multihashes/dist/index.js
